# SmartOps — autonomous incident resolution, with humans in the loop

**TECHgium 2026 national finalist prototype**

SmartOps explores a better incident-management loop for cloud infrastructure: **Observe → Diagnose → Resolve → Review → Learn**.

The problem is familiar to operations teams: alerts, logs, root-cause analysis, and recovery all arrive under pressure. The prototype brings those signals into an AI-assisted workflow while keeping an SRE review point before deployment.

## Rahul's role

Rahul led the AI/backend work: model and pipeline development, training, cloud deployment, and the operating flow behind the proof of concept.

## Built into the proof of concept

- spatial-temporal GNN reasoning for cascading anomalies
- Qwen-assisted diagnostics and remediation suggestions
- Amazon EKS, Terraform, Prometheus, and CloudWatch
- React and D3 operational dashboard
- feedback-based agent iteration before a proposed fix is deployed
