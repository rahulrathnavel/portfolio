# Omni RAG — answers should show their evidence

An answer is only useful when a person can inspect what supports it. Omni RAG is a live multimodal retrieval application designed around that idea.

## Rahul's role

Rahul built the retrieval and evidence-oriented product flow as a public live prototype.

## System choices

- dense retrieval with FAISS
- sparse retrieval with BM25
- multimodal document retrieval
- PDF-page evidence alongside answers
