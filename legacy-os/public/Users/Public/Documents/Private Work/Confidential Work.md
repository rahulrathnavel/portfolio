# Confidential work — scope without names

Some freelance and research work is described only at problem-space level. Client identities, private datasets, and unpublished implementation details stay private.

## Covered areas

- job-signal verification and opportunity matching
- health-research decision support using tabular and image-led inputs; never presented as medical diagnosis
- mobile telemetry for Arduino-connected agricultural sensing
- privacy-oriented smart-grid machine-learning research

The useful rule is simple: show the engineering scope honestly, but never turn a client's work into portfolio decoration without permission.
