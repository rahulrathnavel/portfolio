# Smart Glass Assistant — visual assistance closer to the edge

This accessibility prototype combines wearable visual input with a mobile application for object, text, and face assistance.

## Rahul's role

Rahul built the application and contributed to the system integration work connecting the wearable input and mobile experience.

## Focus

- ESP32-CAM visual input
- Flutter mobile application
- object detection, OCR, and face-recognition flows
- privacy-conscious edge-oriented design

The project is presented as a prototype: useful engineering exploration, not a claim that a demo replaces a production assistive device.
