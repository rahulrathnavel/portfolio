# Raki — a real-time Android chat app

Raki is a native Kotlin and Jetpack Compose chat application built around Firebase authentication, real-time messaging, and MVVM structure.

## Rahul's role

Rahul built the Android client and its maintainable application structure.

## Why it belongs here

A chat app tests whether product behaviour stays understandable as features accumulate: identity, state, messages, delivery, and the small details that make a mobile app feel reliable.
