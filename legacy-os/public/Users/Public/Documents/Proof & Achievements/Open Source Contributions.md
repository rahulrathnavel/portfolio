# Maintainer-merged open source contributions

- [SciPy #25209](https://github.com/scipy/scipy/pull/25209) — safer backend handling for an unknown pytest marker warning.
- [Matplotlib #31707](https://github.com/matplotlib/matplotlib/pull/31707) — prevented a violinplot crash on empty input data.
- [Statsmodels #9812](https://github.com/statsmodels/statsmodels/pull/9812) — corrected residual behaviour for UECM results.
